Tabelas:
    Invite(id, token, email, role, sender_id, expires_at)
    User(id, email, username, password, role)

Rotas chamadas pelo admin:
    POST /users/invite/
        Request:
        {
            "email": <string>,
            "role": <string>
        }

        Na tela de admin, convidar um novo usuário por email, colocando o email de convite e o role
        <url-frontend>/users/invite/?token=<invite_token>

        Passo a passo:
            1. invite_token = uuid4()
            2. Insert Invite(
                token=invite_token,
                email=request.email,
                role=request.role,
                sender_id=current_user.id,
                expires_at=now() + 24h
            )
            3. Enviar email contendo link:
                <url-frontend>/users/invite/?token=<invite_token> #tela frontend, não endpoint!

    DELETE /users/invite/?token=<invite_id>
        Passo a passo:
            1. Procurar no banco algum Invite com token=<invite_token>
            2. Deletar convite

Rotas chamadas pelo novo usuário:
    GET /users/invite/data/?token=<invite_token> #invite token vai estar armazenado no banco
        Response:
        {
            "email": <string>,
            "role": <string>,
            "sender_name": <string>
        }

        Passo a passo:
            1. Procurar no banco algum Invite com token=<invite_token>
            2. Se expires_at < now():
                    Deletar do banco
               Senão:
                    Retornar informações

    POST /users/invite/confirm/?token=<invite_token>
        Request:
        {
            "username": <string>,
            "full_name: <string>,
            "password": <string>
        }

        Passo a passo:
            1. Procurar no banco algum Invite com token=<invite_token> and expires_at < now()
            2. Insert User(
                email=invite.email,
                role=invite.role,
                username=request.username,
                full_name=request.full_name,
                password=bcrypt(request.password)
            )
            3. Deletar Invite
